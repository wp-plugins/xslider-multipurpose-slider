== Description ==

= Overview =

This is the free version of xSlider, a very easy-to-use and full-featured slide plugin for WordPress. It's fully responsive, cross browser, SEO friendly, it uses hardware accelerated CSS3 transforms and it already comes with a lot of ready-to-use animations.

For demos, upgrade to premium version and to a complete list of differences between the free and premium versions, please take a look at xSlider website
[xSlider website](http://xslider.develvet.com/?ref=wordpress)


= Compatible Browsers =

* Firefox

* Chrome

* Android browser



== Installation ==

**Uploading in WordPress Dashboard**

1. Download `xslider-free.zip`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `xslider-free.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard
